The trick to animations is in the syntax

To start using animations we must first learn about the syntax. We are moving into new territory here. Those of you with a little CS experience may be familiar with the following term:


Conditional Statement


A conditional statement is a statement that describes the relationship between two events based on certain conditions. It's also known as an"if-then statement".

conditional statements

but first, let's learn about the animation properties

selector {

animation-name: this is where the animation gets called, ie part 2

animation-delay: delays start time

animation-direction: your animation can run forwards or backwards

animation-duration: specifies the amount of time it takes to complete one cycle

animation-fill-mode: styles the element for before and after the animation

animation-iteration-count: how many times will it play?

animation-play-state: is it running or paused? you can use a psuedo class to trigger an animation

animation-timing-functionspeed curve ie ease-in, linear et al

}
