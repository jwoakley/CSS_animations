How to Use CSS Transitions

Transitions are properties that help us manipulate elemental change in CSS - These are REACTIVE, they need to be triggered. For a transition to work you need to specify at least two things:


• the property you want to affect

• the duration of the effect


We actually already started using transitions by adding a duration to our transormations, right?

The Transition Properties are:


transition-property:

transition-duration:

transition-timing-function:

transition-delay:


They are always written out in this order. Get in the habit of calling all four for each transition

transition-property

selector {

transition-property: opacity;

transition-duration:

transition-timing-function:

transition-delay:

}

with the transition-property declaration we can target all the properties, or get specific, for example:


• color

• opacity

• height

• width

you can pretty much manipulate any property!!! if you want to add more than one property, just seperate by a comma. for example:

transition-property: opacity, width;

transition-duration

We are already experts at this one! the duration is simply how much time we want the transition to last

selector {

transition-property: opacity;

transition-duration: 5s

transition-timing-function:

transition-delay:

}

transition-timing-function

selector {

transition-property: opacity;

transition-duration: 5s

transition-timing-function: ease-out;

transition-delay:

}

ease functions: broken down

Ease functions may seem a little overwhelming at first, but once you understand the basics you will understand their value.
We use these to add further nuance to our transitions (this is how we get a screen that fades to black).

  • linear: DEFAULT a constant speed
  • ease: speeds up a little in the middle, then slows down again
  • ease-in: starts off slow and accelerates towrds the end
  • ease-out: starts off fast and slows to the end
  • ease-in-out: like a doppler, slow start, fast middle, slow end

transition-delay

This is an easy one, we can add a delay to start measured in seconds or miliseconds

selector {

transition-property: opacity;

transition-duration: 5s;

transition-timing-function: ease-out;

transition-delay: 2s;

}
